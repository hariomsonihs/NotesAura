## Formatting Guide

### Text Formatting

- Use **double asterisks** for bold text
- Use *single asterisks* for italic text
- Use `backticks` for inline code (not fully supported yet)

### Headings

```
# Main Heading (H1)
## Sub Heading (H2)
### Smaller Heading (H3)
```

### Lists

Bullet points:
- Item 1
- Item 2
- Item 3

Numbered lists:
1. First item
2. Second item
3. Third item

### Code Blocks

Use triple backticks for code blocks:

```
```language
your code here
```
```

### Quotes

> Use greater-than symbol for quotes or important notes

### Horizontal Lines

Use three dashes for horizontal dividers:

## Best Practices

1. Keep paragraphs short and readable
2. Use headings to organize content
3. Add code examples where relevant
4. Include images to illustrate concepts
5. Use quotes for important notes

**Remember**: The app will automatically format your content beautifully!

## Image URLs

For images, you can use:
- Direct image URLs (https://example.com/image.jpg)
- GitHub raw URLs (https://raw.githubusercontent.com/...)
- Any publicly accessible image URL

Make sure images are:
- Publicly accessible
- In common formats (JPG, PNG, GIF)
- Reasonably sized for mobile viewing
