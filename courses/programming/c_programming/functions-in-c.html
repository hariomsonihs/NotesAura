Functions in C

What are Functions?

A function in C is a block of code that performs a specific task. Functions allow reusability, modularity, and better organization of programs.

  • Prevents repetition of code.
  • Makes programs easier to read and maintain.
  • Can be reused across multiple programs.

Types of Functions

  • Library Functions: Predefined (e.g., printf(), sqrt()).
  • User-defined Functions: Created by the programmer.

Syntax of Function

return_type function_name(parameters) {
    // body of function
}

Example: User-defined Function

#include <stdio.h>

int add(int a, int b) {  
    return a + b;  
}

int main() {
    int result = add(5, 3);
    printf("Sum = %d\n", result);
    return 0;
}

Function Categories

  • Without arguments and without return value
  • With arguments and without return value
  • Without arguments and with return value
  • With arguments and with return value

Example: Function with Arguments and Return Value

#include <stdio.h>

int multiply(int x, int y) {
    return x * y;
}

int main() {
    printf("Product = %d\n", multiply(4, 6));
    return 0;
}

Example: Recursive Function (Factorial)

#include <stdio.h>

int factorial(int n) {
    if(n == 0) 
        return 1;
    else 
        return n * factorial(n - 1);
}

int main() {
    printf("Factorial of 5 = %d\n", factorial(5));
    return 0;
}

Common Mistakes

  • Not declaring a function before using it.
  • Mismatching return type with actual returned value.
  • Forgetting to include necessary header files.
Conclusion:

Functions in C make programs modular and reusable. By understanding function categories, return types, and recursion, programmers can write efficient and structured code. Mastering functions is essential for solving real-world problems in C.
