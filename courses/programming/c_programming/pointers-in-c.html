Pointers in C

What are Pointers?

A pointer is a variable that stores the memory address of another variable. Instead of holding actual data, it holds the location of the data in memory.

  • Declaration: int *ptr; creates a pointer to int.
  • Address-of Operator (&): Used to get the memory address of a variable.
  • Dereference Operator (*): Accesses the value stored at the address.

Basic Pointer Example

#include <stdio.h>

int main() {
    int x = 10;
    int *p = &x;

    printf("Value of x: %d\n", x);
    printf("Address of x: %p\n", &x);
    printf("Pointer p stores: %p\n", p);
    printf("Value pointed by p: %d\n", *p);

    return 0;
}

Pointer Arithmetic

Pointers support arithmetic operations like increment, decrement, addition, and subtraction. These move the pointer by the size of the data type.

  • p++: Moves pointer to next element of its type.
  • p--: Moves pointer to previous element.
  • p + n: Moves pointer n elements forward.

Pointers with Arrays

Array names act as constant pointers. Traversing arrays using pointers is common.

int arr[3] = {10, 20, 30};
int *p = arr;

for(int i=0; i<3; i++) {
    printf("%d ", *(p+i));
}

Double Pointers

A pointer to a pointer stores the address of another pointer, useful in dynamic memory and multi-level references.

int x = 5;
int *p = &x;
int **pp = &p;

printf("x = %d", **pp);

Function Pointers

Pointers can store function addresses, enabling callbacks and dynamic execution.

void greet() {
    printf("Hello!");
}

int main() {
    void (*fp)() = greet;
    fp();
}

Pointers and Dynamic Memory

Pointers are essential in dynamic allocation functions like malloc, calloc, realloc, and free.

int *arr = (int*) malloc(5 * sizeof(int));
for(int i=0; i<5; i++) arr[i] = i+1;

free(arr);

Common Mistakes with Pointers

  • Dereferencing uninitialized or NULL pointers → segmentation fault.
  • Memory leaks when forgetting to call free().
  • Pointer arithmetic beyond array bounds → undefined behavior.
Conclusion:

Pointers provide direct memory access and powerful features in C programming. Mastery of pointers is essential for dynamic memory, arrays, strings, and efficient low-level coding.
