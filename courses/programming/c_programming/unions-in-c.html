Unions in C

What is a Union?

A union in C is a special data type that allows storing different data types in the same memory location. Unlike structures, all members of a union share the same memory. This means at any given time, only one member can hold a valid value.

  • Efficient Memory Usage: Union size equals the size of its largest member.
  • One Active Member: Only one field is usable at a time.
  • Declared with: union keyword.

Union Declaration Example

union Data {
    int i;
    float f;
    char str[20];
};

int main() {
    union Data d;
    d.i = 10;
    printf("d.i = %d\n", d.i);
    d.f = 220.5;
    printf("d.f = %.2f\n", d.f);
    strcpy(d.str, "C Language");
    printf("d.str = %s\n", d.str);
    return 0;
}

Output: Only the last assigned member (string) will have a valid value because all share the same memory.

Difference Between Structures and Unions

  • Structures: Each member has its own storage; total size is sum of all members.
  • Unions: All members share one storage; size equals largest member.
  • Structures: Multiple members usable at the same time.
  • Unions: Only one member can be used at a time.

Memory Layout of Union

For the above example:

Union Data: size = max(sizeof(int), sizeof(float), sizeof(char[20])) = 20 bytes.

All members i, f, and str share the same 20 bytes.

Accessing Union Members

  • Use . (dot operator) with variable.
  • Use -> (arrow operator) with pointer.

Union with Pointer

union Data {
    int i;
    float f;
};

int main() {
    union Data d, *ptr;
    ptr = &d;
    ptr->i = 25;
    printf("Value of i = %d", ptr->i);
    return 0;
}
Use Cases of Unions:
  • Efficient memory handling in embedded systems.
  • Storing data in multiple formats (e.g., same bytes interpreted as int or float).
  • Implementing variant data structures.
Conclusion:

Unions in C are useful for memory-efficient programming when only one member is needed at a time. While powerful, they must be used carefully to avoid overwriting data unintentionally.
