Looping in C Programming Language

Introduction to Looping

Looping in C allows a set of instructions to be executed repeatedly based on a condition or a fixed number of iterations. Loops are essential for automating repetitive tasks and processing collections of data.

  • Definition: Loops are control structures that execute a block of code multiple times until a condition is met or a specified number of iterations is completed.
  • Key Points:
    • Loops reduce code redundancy by automating repetitive tasks.
    • They rely on conditions, counters, or expressions to control execution.
    • C provides several looping constructs for different use cases.

Types of Loops in C

C supports three primary looping constructs, each designed for specific scenarios, along with control statements to modify loop behavior.

  • Definition: Loop types are constructs that facilitate repeated execution of code blocks.
  • Types:
    • for: Ideal for loops with a known number of iterations.
    • while: Executes as long as a condition is true.
    • do-while: Executes at least once, then continues if the condition is true.
  • Control Statements:
    • break: Exits the loop immediately.
    • continue: Skips the current iteration and proceeds to the next.
Note: Choose the loop type based on whether the number of iterations is known and whether the loop needs to execute at least once.

The for Loop

The for loop is used when the number of iterations is known in advance, typically involving a counter.

  • Syntax: 
    for (initialization; condition; update) {
    // Code to execute repeatedly
}
  • Key Points:
    • Initialization: Executed once before the loop starts (e.g., int i = 0).
    • Condition: Checked before each iteration; loop continues if true.
    • Update: Executed after each iteration (e.g., i++).
    • All three parts are optional, but semicolons are required.

Example: for Loop

#include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        printf("Iteration %d\\n", i);
    }
    return 0;
}
  • for: Iterates from i=1 to i=5.
  • Output: "Iteration 1" to "Iteration 5".

The while Loop

The while loop executes a block of code as long as a condition is true, suitable when the number of iterations is unknown.

  • Syntax: 
    while (condition) {
    // Code to execute repeatedly
}
  • Key Points:
    • Condition is checked before each iteration.
    • Loop may not execute if the condition is initially false.
    • Ensure the condition eventually becomes false to avoid infinite loops.

Example: while Loop

#include <stdio.h>
int main() {
    int i = 1;
    while (i <= 5) {
        printf("Iteration %d\\n", i);
        i++;
    }
    return 0;
}
  • while: Continues until i exceeds 5.
  • Output: "Iteration 1" to "Iteration 5".

The do-while Loop

The do-while loop executes a block of code at least once before checking the condition.

  • Syntax: 
    do {
    // Code to execute repeatedly
} while (condition);
  • Key Points:
    • Condition is checked after each iteration.
    • Guaranteed to execute at least once.
    • Useful for scenarios requiring initial execution (e.g., user input validation).

Example: do-while Loop

#include <stdio.h>
int main() {
    int i = 1;
    do {
        printf("Iteration %d\\n", i);
        i++;
    } while (i <= 5);
    return 0;
}
  • do-while: Executes code, then checks if i <= 5.
  • Output: "Iteration 1" to "Iteration 5".

Control Statements: break and continue

Control statements modify the flow of loops, allowing early termination or skipping iterations.

  • break: Exits the loop entirely, regardless of the condition.
  • continue: Skips the current iteration and proceeds to the next.
  • Key Points:
    • break is useful for stopping a loop when a specific condition is met.
    • continue skips remaining code in the current iteration.
    • Both can be used in all loop types and switch statements.

Example: break and continue

#include <stdio.h>
int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 3) {
            continue; // Skip iteration when i is 3
        }
        if (i == 7) {
            break; // Exit loop when i is 7
        }
        printf("Iteration %d\\n", i);
    }
    return 0;
}
  • continue: Skips printing when i is 3.
  • break: Stops loop when i is 7.
  • Output: "Iteration 1", "Iteration 2", "Iteration 4", "Iteration 5", "Iteration 6".

Nested Loops

Nested loops involve placing one loop inside another, useful for processing multi-dimensional data or complex iterations.

  • Definition: A loop within another loop, where the inner loop completes all iterations for each iteration of the outer loop.
  • Key Points:
    • Commonly used for tasks like matrix operations or pattern printing.
    • Can impact performance if deeply nested; optimize where possible.
    • break and continue affect only the innermost loop unless labeled (non-standard in C).

Example: Nested Loops

#include <stdio.h>
int main() {
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 2; j++) {
            printf("i=%d, j=%d\\n", i, j);
        }
    }
    return 0;
}
  • Nested for: Inner loop runs twice for each outer loop iteration.
  • Output: Prints pairs (i=1, j=1), (i=1, j=2), (i=2, j=1), (i=2, j=2), (i=3, j=1), (i=3, j=2).

Infinite Loops

An infinite loop runs indefinitely unless terminated externally or with a break statement.

  • Definition: A loop that never terminates naturally due to a condition that remains true.
  • Key Points:
    • Can be intentional (e.g., in embedded systems) or accidental (e.g., due to logic errors).
    • Commonly created with for(;;) or while(1).
    • Use break or external conditions to exit intentional infinite loops.

Example: Infinite Loop with break

#include <stdio.h>
int main() {
    int count = 0;
    while (1) {
        count++;
        printf("Count: %d\\n", count);
        if (count == 5) {
            break;
        }
    }
    return 0;
}
  • while(1): Creates an infinite loop.
  • break: Exits when count reaches 5.
  • Output: "Count: 1" to "Count: 5".

Loop Control with goto (Advanced)

The goto statement can be used to jump to a labeled statement, but it is rarely recommended for loop control.

  • Syntax: 
    label:
    // Code
goto label;
  • Key Points:
    • Avoid goto in favor of structured loops for clarity.
    • Used in specific cases like error handling in low-level programming.
    • Can make code harder to maintain and debug.

Example: goto in Loop

#include <stdio.h>
int main() {
    int i = 1;
start:
    if (i > 5) {
        goto end;
    }
    printf("Iteration %d\\n", i);
    i++;
    goto start;
end:
    return 0;
}
  • goto: Jumps to start or end labels.
  • Output: "Iteration 1" to "Iteration 5".
  • Note: Structured loops are preferred over goto.
