Operators in C Programming Language

Introduction to Operators

Operators in C are special symbols or keywords used to perform operations on variables and values. They are essential for manipulating data, controlling program flow, and performing computations in C programs.

  • Definition: Operators are symbols that instruct the compiler to perform specific mathematical, logical, or relational operations.
  • Key Points:
    • Operators work on operands (variables or values) to produce a result.
    • They are categorized based on their functionality, such as arithmetic, relational, or logical.
    • Operators have precedence and associativity rules that determine the order of evaluation.

Types of Operators

C provides a rich set of operators, each serving a specific purpose. They can be classified into several categories based on their functionality.

  • Definition: Operator types classify operators based on the operations they perform.
  • Categories:
    • Arithmetic Operators: Perform mathematical operations (e.g., +, -, *, /, %).
    • Relational Operators: Compare two values (e.g., ==, !=, >, <).
    • Logical Operators: Perform logical operations (e.g., &&, ||, !).
    • Bitwise Operators: Operate on bits (e.g., &, |, ^, ~, <<, >>).
    • Assignment Operators: Assign values to variables (e.g., =, +=, -=).
    • Increment/Decrement Operators: Increase or decrease a variable’s value (e.g., ++, --).
    • Conditional (Ternary) Operator: A shorthand for if-else (?:).
    • Special Operators: Include operators like sizeof, & (address), * (dereference).
Note: Operator precedence determines the order of evaluation in expressions. Use parentheses to override default precedence for clarity.

Arithmetic Operators

Arithmetic operators perform basic mathematical operations on numeric operands.

  • Operators:
    • +: Addition
    • -: Subtraction
    • *: Multiplication
    • /: Division
    • %: Modulus (remainder)

Example: Arithmetic Operators

#include <stdio.h>
int main() {
    int a = 10, b = 3;
    printf("Addition: %d\\n", a + b); // 13
    printf("Subtraction: %d\\n", a - b); // 7
    printf("Multiplication: %d\\n", a * b); // 30
    printf("Division: %d\\n", a / b); // 3
    printf("Modulus: %d\\n", a % b); // 1
    return 0;
}
  • +: Adds a and b.
  • /: Integer division, truncates decimal part.
  • %: Returns remainder of a divided by b.

Relational Operators

Relational operators compare two values and return a boolean result (0 for false, 1 for true).

  • Operators:
    • ==: Equal to
    • !=: Not equal to
    • >: Greater than
    • <: Less than
    • >=: Greater than or equal to
    • <=: Less than or equal to

Example: Relational Operators

#include <stdio.h>
int main() {
    int x = 5, y = 10;
    printf("x == y: %d\\n", x == y); // 0 (false)
    printf("x != y: %d\\n", x != y); // 1 (true)
    printf("x > y: %d\\n", x > y); // 0 (false)
    printf("x < y: %d\\n", x < y); // 1 (true)
    return 0;
}
  • ==: Checks equality of x and y.
  • !=: Checks if x and y are different.

Logical Operators

Logical operators perform logical operations and are used in decision-making constructs.

  • Operators:
    • &&: Logical AND
    • ||: Logical OR
    • !: Logical NOT

Example: Logical Operators

#include <stdio.h>
int main() {
    int a = 1, b = 0;
    printf("a && b: %d\\n", a && b); // 0 (false)
    printf("a || b: %d\\n", a || b); // 1 (true)
    printf("!a: %d\\n", !a); // 0 (false)
    return 0;
}
  • &&: True if both operands are true.
  • ||: True if at least one operand is true.
  • !: Inverts the truth value.

Bitwise Operators

Bitwise operators manipulate individual bits of operands, often used in low-level programming.

  • Operators:
    • &: Bitwise AND
    • |: Bitwise OR
    • ^: Bitwise XOR
    • ~: Bitwise NOT
    • <<: Left shift
    • >>: Right shift

Example: Bitwise Operators

#include <stdio.h>
int main() {
    int a = 5, b = 3; // a: 0101, b: 0011
    printf("a & b: %d\\n", a & b); // 1 (0001)
    printf("a | b: %d\\n", a | b); // 7 (0111)
    printf("a ^ b: %d\\n", a ^ b); // 6 (0110)
    printf("~a: %d\\n", ~a); // -6
    printf("a << 1: %d\\n", a << 1); // 10 (1010)
    return 0;
}
  • &: Sets bits to 1 where both operands have 1.
  • <<: Shifts bits left, filling with zeros.

Assignment Operators

Assignment operators assign values to variables, often combining with other operations.

  • Operators:
    • =: Simple assignment
    • +=, -=, *=, /=, %=: Compound assignment
    • &=, |=, ^=, <<=, >>=: Bitwise compound assignment

Example: Assignment Operators

#include <stdio.h>
int main() {
    int x = 10;
    x += 5; // x = x + 5
    printf("x += 5: %d\\n", x); // 15
    x *= 2; // x = x * 2
    printf("x *= 2: %d\\n", x); // 30
    return 0;
}
  • +=: Adds and assigns the result.
  • *=: Multiplies and assigns the result.

Increment and Decrement Operators

Increment and decrement operators increase or decrease a variable’s value by 1.

  • Operators:
    • ++: Increment (prefix and postfix)
    • --: Decrement (prefix and postfix)

Example: Increment and Decrement Operators

#include <stdio.h>
int main() {
    int a = 5;
    printf("a: %d\\n", a); // 5
    printf("++a: %d\\n", ++a); // 6 (prefix)
    printf("a++: %d\\n", a++); // 6 (postfix, prints 6, then a becomes 7)
    printf("a: %d\\n", a); // 7
    return 0;
}
  • ++a: Increments before use.
  • a++: Uses value, then increments.

Conditional (Ternary) Operator

The ternary operator is a shorthand for an if-else statement.

  • Syntax: condition ? expression1 : expression2
  • Key Points:
    • Evaluates condition; returns expression1 if true, else expression2.
    • Useful for concise conditional assignments.
