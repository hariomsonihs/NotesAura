Arrays in C Programming Language

Introduction to Arrays

Arrays in C are used to store multiple values of the same data type in a single variable, accessed using indices. They are essential for managing collections of data efficiently.

  • Definition: An array is a collection of elements of the same data type stored in contiguous memory locations.
  • Key Points:
    • Arrays allow efficient storage and access to multiple values.
    • Each element is accessed using an index, starting from 0.
    • Arrays are fixed in size once declared, unless dynamically allocated.

Types of Arrays

Arrays in C can be classified based on their dimensions and usage.

  • Definition: Array types categorize arrays based on the number of dimensions and data they store.
  • Types:
    • One-Dimensional Arrays: A single row of elements (e.g., int arr[5]).
    • Multi-Dimensional Arrays: Arrays with multiple dimensions, like 2D (matrix) or 3D arrays (e.g., int matrix[3][4]).
    • Dynamic Arrays: Allocated at runtime using pointers and memory allocation functions like malloc.
Note: Choose the array type based on the data structure needed; use dynamic arrays for flexible sizes.

Rules for Declaring Arrays

Arrays must follow specific rules to ensure valid declaration and usage in C.

  • Definition: Declaration rules specify how arrays are defined, named, and sized.
  • Key Points:
    • Syntax: data_type array_name[size] (e.g., int numbers[10]).
    • Array names follow identifier naming rules (start with a letter or underscore, no keywords).
    • Size must be a positive integer or constant expression at compile time for static arrays.
    • Arrays are zero-indexed; valid indices range from 0 to size-1.
    • Uninitialized arrays contain garbage values unless explicitly initialized.

Example: Array Declaration

#include <stdio.h>
int main() {
    int numbers[5]; // Declaration of a 5-element integer array
    numbers[0] = 10; // Assigning value to first element
    printf("First element: %d\\n", numbers[0]);
    return 0;
}
  • numbers[5]: Declares an array of 5 integers.
  • numbers[0]: Accesses the first element.
  • Output: "First element: 10".

Initializing Arrays

Arrays can be initialized at declaration or later using assignment. Initialization sets the values of array elements.

  • Syntax: 
    data_type array_name[size] = {value1, value2, ..., valueN};
  • Key Points:
    • Partial initialization: Unspecified elements are set to 0.
    • Size can be omitted if initialized at declaration (e.g., int arr[] = {1, 2, 3}).
    • Individual elements can be assigned using indices (e.g., arr[0] = 5).
    • Initialization at declaration is preferred for static arrays.

Example: Array Initialization

#include <stdio.h>
int main() {
    int arr[5] = {1, 2, 3}; // Partial initialization, arr[3] and arr[4] are 0
    int arr2[] = {10, 20, 30, 40}; // Size inferred as 4
    for (int i = 0; i < 5; i++) {
        printf("arr[%d] = %d\\n", i, arr[i]);
    }
    return 0;
}
  • arr[5]: Initializes first three elements, others are 0.
  • arr2[]: Size is automatically set to 4.
  • Output: "arr[0] = 1", "arr[1] = 2", "arr[2] = 3", "arr[3] = 0", "arr[4] = 0".

Accessing Array Elements

Array elements are accessed using their index, which specifies their position in the array.

  • Syntax: array_name[index]
  • Key Points:
    • Indices start at 0 and go up to size-1.
    • Accessing out-of-bounds indices causes undefined behavior.
    • Arrays can be accessed in loops for efficient processing.

Example: Accessing Array Elements

#include <stdio.h>
int main() {
    int arr[4] = {10, 20, 30, 40};
    for (int i = 0; i < 4; i++) {
        printf("Element at index %d: %d\\n", i, arr[i]);
    }
    return 0;
}
  • arr[i]: Accesses elements using loop index.
  • Output: "Element at index 0: 10" to "Element at index 3: 40".

Multi-Dimensional Arrays

Multi-dimensional arrays store data in multiple dimensions, such as rows and columns for 2D arrays.

  • Syntax: 
    data_type array_name[rows][columns]; // 2D array
  • Key Points:
    • Commonly used for matrices, tables, or grids.
    • Requires nested loops to access or process elements.
    • Memory is still contiguous, stored row-major order in C.

Example: 2D Array

#include <stdio.h>
int main() {
    int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("matrix[%d][%d] = %d\\n", i, j, matrix[i][j]);
        }
    }
    return 0;
}
  • matrix[2][3]: 2 rows, 3 columns.
  • Nested loops: Access each element.
  • Output: Prints all elements in row-major order.

Dynamic Arrays (Advanced)

Dynamic arrays are allocated at runtime using memory allocation functions, allowing flexible sizing.

  • Syntax: 
    data_type *array = (data_type*)malloc(size * sizeof(data_type));
  • Key Points:
    • Use malloc, calloc, or realloc for allocation.
    • Free memory using free to prevent memory leaks.
    • Dynamic arrays are accessed like static arrays but require pointer arithmetic.

Example: Dynamic Array

#include <stdio.h>
#include <stdlib.h>
int main() {
    int *arr = (int*)malloc(5 * sizeof(int));
    if (arr == NULL) {
        printf("Memory allocation failed\\n");
        return 1;
    }
    for (int i = 0; i < 5; i++) {
        arr[i] = i + 1;
        printf("arr[%d] = %d\\n", i, arr[i]);
    }
    free(arr);
    return 0;
}
  • malloc: Allocates memory for 5 integers.
  • free: Releases allocated memory.
  • Output: "arr[0] = 1" to "arr[4] = 5".

Arrays and Functions

Arrays can be passed to functions, typically as pointers, to operate on their elements.

  • Syntax: 
    void function_name(data_type array[], int size);
  • Key Points:
    • Arrays decay to pointers when passed to functions.
    • Pass the array size explicitly to avoid out-of-bounds errors.
    • Modifications to array elements in functions affect the original array.

Example: Array in Function

#include <stdio.h>
void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("Element %d: %d\\n", i, arr[i]);
    }
}
int main() {
    int arr[3] = {10, 20, 30};
    printArray(arr, 3);
    return 0;
}
  • printArray: Takes array and size as parameters.
  • Output: "Element 0: 10" to "Element 2: 30".

Common Operations on Arrays

Arrays support various operations like traversal, searching, sorting, and manipulation.

  • Operations:
    • Traversal: Accessing each element, typically with a loop.
    • Searching: Finding an element (e.g., linear or binary search).
    • Sorting: Arranging elements in order (e.g., bubble sort, quicksort).
    • Insertion/Deletion: Adding or removing elements (complex in static arrays).
