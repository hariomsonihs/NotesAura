Tokens in C Programming Language

Introduction to Tokens

Tokens in C are the smallest individual units in a program that the compiler recognizes. They form the building blocks of C source code.

  • Definition: A token is the smallest element of a C program that carries meaning for the compiler.
  • Key Points:
    • C programs are composed of tokens, which are processed by the compiler during lexical analysis.
    • Tokens include keywords, identifiers, constants, operators, and special symbols.
    • Understanding tokens is essential for writing syntactically correct C programs.

Types of Tokens in C

C tokens are categorized into six main types, each serving a specific purpose in the language.

  • Definition: Tokens are classified based on their role in the program, such as defining operations, naming variables, or specifying constants.
  • Key Points:
    • Keywords: Reserved words with predefined meanings (e.g., int, if, return).
    • Identifiers: User-defined names for variables, functions, or other entities.
    • Constants: Fixed values that cannot be modified (e.g., integers, floating-point numbers, characters).
    • Operators: Symbols that perform operations (e.g., +, -, *, /).
    • Special Symbols: Punctuation marks like braces, commas, and semicolons.
    • Strings: Sequences of characters enclosed in double quotes.

Example: Identifying Tokens

#include <stdio.h> // Preprocessor directive

int main() {
    int num = 10; // Tokens: int, num, =, 10, ;
    printf("Value: %d\\n", num); // Tokens: printf, (, "Value: %d\\n", ,, num, ), ;
    return 0; // Tokens: return, 0, ;
}
  • int: Keyword token.
  • num: Identifier token.
  • =: Operator token.
  • 10: Constant token.
  • ;: Special symbol token.

Keywords

Keywords are reserved words in C that have special meanings and cannot be used as identifiers.

  • Definition: Keywords are predefined words reserved by the C language for specific purposes.
  • Key Points:
    • C has 32 keywords in the C89 standard, with additional ones in C99 and C11 (e.g., _Bool, _Complex).
    • Examples: auto, break, case, const, do, else, for, if, return, void.
    • Keywords are case-sensitive and must be written in lowercase.
Note: Avoid using keywords as variable or function names to prevent compilation errors.

Identifiers

Identifiers are user-defined names given to variables, functions, and other entities in a C program.

  • Definition: An identifier is a name used to identify a variable, function, or any other user-defined item.
  • Key Points:
    • Can contain letters (a-z, A-Z), digits (0-9), and underscores (_).
    • Must start with a letter or underscore.
    • Case-sensitive (e.g., count and Count are different).
    • Should be descriptive for better code readability.

Example: Valid and Invalid Identifiers

#include <stdio.h>

int main() {
    int student_count = 50; // Valid identifier
    float averageMarks = 85.5; // Valid identifier
    char _grade = 'A'; // Valid identifier
    // int 2marks = 90; // Invalid: starts with a digit
    // int for = 10; // Invalid: uses a keyword
    printf("Count: %d, Marks: %.1f, Grade: %c\\n", student_count, averageMarks, _grade);
    return 0;
}
  • student_count: Valid identifier, descriptive and follows rules.
  • 2marks: Invalid, starts with a digit.
  • for: Invalid, uses a reserved keyword.

Constants

Constants are fixed values that cannot be modified during program execution.

  • Definition: Constants represent immutable values used in a program.
  • Key Points:
    • Types include integer constants (e.g., 42), floating-point constants (e.g., 3.14), character constants (e.g., 'A'), and string constants (e.g., "Hello").
    • Can be defined using const keyword or #define directive.
    • Constants improve code safety and readability.

Example: Using Constants

#include <stdio.h>
#define MAX_STUDENTS 100 // Macro constant

int main() {
    const float PI = 3.14159; // Constant variable
    int total = MAX_STUDENTS;
    printf("Max Students: %d, PI: %.5f\\n", total, PI);
    return 0;
}
  • #define MAX_STUDENTS: Macro constant for fixed value.
  • const float PI: Constant variable, unmodifiable.

Operators

Operators are symbols that perform operations on variables and values.

  • Definition: Operators specify computations like arithmetic, logical, or relational operations.
  • Key Points:
    • Arithmetic Operators: +, -, *, /, % (modulus).
    • Relational Operators: ==, !=, >, <, >=, <=.
    • Logical Operators: &&, ||, !.
    • Bitwise Operators: &, |, ^, ~, <<, >>.
    • Assignment Operators: =, +=, -=, *=, /=, etc.
    • Other Operators: sizeof, & (address), * (pointer), etc.

Example: Using Operators

#include <stdio.h>

int main() {
    int a = 10, b = 5;
    int sum = a + b; // Arithmetic operator
    int isEqual = (a == b); // Relational operator
    int bitwiseAnd = a & b; // Bitwise operator
    printf("Sum: %d, Equal: %d, Bitwise AND: %d\\n", sum, isEqual, bitwiseAnd);
    return 0;
}
  • +: Arithmetic addition.
  • ==: Relational equality check.
  • &: Bitwise AND operation.

Special Symbols

Special symbols are punctuation marks and other characters used to structure C programs.

  • Definition: Special symbols include braces, parentheses, semicolons, and other characters with syntactic roles.
  • Key Points:
    • Braces {}: Define blocks of code.
    • Parentheses (): Used in function calls and expressions.
    • Semicolon ;: Terminates statements.
    • Comma ,: Separates elements in lists.
    • Other Symbols: #, [], *, &, etc.

Example: Special Symbols

#include <stdio.h>

int main() {
    int arr[2] = {10, 20}; // Braces, brackets, comma
    printf("Array: %d, %d\\n", arr[0], arr[1]); // Parentheses, comma, semicolon
    return 0;
}
  • {}: Defines the array initialization block.
  • []: Array indexing.
  • ,: Separates array elements.
  • ;: Terminates the statement.

Strings

Strings in C are sequences of characters stored as arrays and terminated by a null character.

  • Definition: A string is a sequence of characters enclosed in double quotes, ending with a null character (\0).
  • Key Points:
    • Declared as char arrays or pointers (e.g., char str[] = "Hello").
    • Strings are manipulated using standard library functions like strlen, strcpy, etc.
    • The null character (\0) marks the end of a string.

Example: Using Strings

#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello, World!";
    printf("String: %s, Length: %zu\\n", str, strlen(str));
    return 0;
}
  • "Hello, World!": String constant with null terminator.
  • strlen: Calculates string length, excluding \0.

Advanced Token Concepts

Advanced understanding of tokens involves their role in preprocessing, lexical analysis, and optimization.

  • Definition: Advanced token concepts relate to how tokens are processed by the compiler and preprocessor.
  • Key Points:
    • Preprocessing Tokens: Tokens processed by the preprocessor (e.g., #define, #include).
    • Token Pasting: Combining tokens using the ## operator in macros.
    • Stringification: Converting tokens to strings using the # operator in macros.
    • Token Scope: Identifiers have specific scopes (local, global) affecting their visibility.
