Structures in C

What is a Structure?

A Structure in C is a user-defined data type that groups different types of variables under one name. Unlike arrays, which hold elements of the same type, structures can hold variables of different data types.

  • Combines related data into a single unit.
  • Each variable inside a structure is called a member.
  • Provides better data organization, especially for complex entities like student, employee, etc.

Declaring and Defining a Structure

struct Student {
    int id;
    char name[50];
    float marks;
};

int main() {
    struct Student s1 = {1, "Alice", 89.5};
    printf("ID: %d, Name: %s, Marks: %.2f\n", s1.id, s1.name, s1.marks);
    return 0;
}

Accessing Members

  • Dot Operator (.): Used to access members directly.
  • Arrow Operator (->): Used with pointers to structures.
struct Student *ptr = &s1;
printf("Name: %s", ptr->name);

Nested Structures

struct Address {
    char city[50];
    int pincode;
};

struct Employee {
    int id;
    char name[50];
    struct Address addr; // nested structure
};

Array of Structures

struct Student students[3];

students[0].id = 1;
strcpy(students[0].name, "Alice");
students[0].marks = 85.0;

Structures with Functions

void display(struct Student s) {
    printf("ID: %d, Name: %s, Marks: %.2f\n", s.id, s.name, s.marks);
}

Structure vs Union

  • Structure: Allocates separate memory for each member.
  • Union: Shares memory among all members, only one can be used at a time.
Key Points:
  • Structures are stored in contiguous memory.
  • Pointers with structures make them powerful for dynamic data handling.
  • Widely used in real-life projects like database records and system programming.
Conclusion:

Structures in C provide a flexible way of grouping related data of different types into a single unit. They are foundational for building complex data structures like linked lists, trees, and graphs, making them essential for C programming mastery.
