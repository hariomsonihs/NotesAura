Login Page Project

Project Overview

This project is a Modern Login Page built using HTML, CSS, and JavaScript. It features a sleek and professional design with interactive elements and animations. The login page includes the following features:

  • Modern and Professional Design: Gradient background with a dark theme and glass morphism effect on the login box.
  • Coding-Themed Animations: Floating coding icons ({ }, , #, console.log();, etc.) and terminal-like text animation.
  • Interactive Login Form: Email and password input fields with real-time validation and focus effects.
  • Funny and Interactive Button Behavior: Sign In button shakes and displays an error message if clicked without filling details.
  • Success Feedback: Displays a success message with a bouncing checkmark icon upon successful submission.
  • Responsive Design: Works seamlessly on all devices (desktop, tablet, mobile).
  • Hover and Transition Effects: Smooth hover effects on buttons and links, and transitions for input fields and login box.
The login page is designed to be responsive and provides a dynamic user experience.

Requirements

  • Programming Language: HTML, CSS, JavaScript
  • Frameworks/Libraries: None
  • Tools: VS Code, Git
  • APIs: None
  • Other Dependencies: Font Awesome (for icons)

Process/Steps

  1. Step 1: Create Project Files
    • Create a folder named login-page-project.
    • Inside the folder, create three files:
      • index.html - For the HTML structure.
      • styles.css - For styling the login page.
      • script.js - For handling form validation and interactivity.
  2. Step 2: Write HTML Code
    • In index.html, create the basic structure of the login page using HTML.
    • Add a login form with email and password fields, a Sign In button, and a registration link.
  3. Step 3: Style the Login Page
    • In styles.css, style the login page to make it visually appealing.
    • Use gradients, glass morphism effects, and animations for a modern look.
  4. Step 4: Add JavaScript Logic
    • In script.js, write functions to handle form validation, button behavior, and dynamic messages.
  5. Step 5: Test and Deploy
    • Test the login page on different devices to ensure responsiveness.
    • Deploy the project using GitHub Pages or any hosting service.

JavaScript Code


document.querySelector('form').addEventListener('submit', function (e) {
    e.preventDefault(); // Prevent form submission

    const email = document.getElementById('email').value;
    const password = document.getElementById('password').value;
    const signInButton = document.querySelector('button');

    if (!email || !password) {
        // Add funny animation to the button
        signInButton.style.position = 'relative';
        signInButton.style.animation = 'shake 0.5s ease-in-out';

        // Show a message on the screen
        const message = document.createElement('div');
        message.classList.add('error-message');
        message.textContent = 'Fill the details before!';
        document.querySelector('.login-box').appendChild(message);

        // Remove the message after 2 seconds
        setTimeout(() => {
            message.remove();
        }, 2000);

        // Remove the shake animation after it ends
        signInButton.addEventListener('animationend', () => {
            signInButton.style.animation = '';
        });
    } else {
        // Simulate successful login
        const successMessage = document.createElement('div');
        successMessage.classList.add('success-message');
        successMessage.innerHTML = ` Sign In Successful!`;
        document.querySelector('.login-box').appendChild(successMessage);

        // Show the success message
        successMessage.style.display = 'block';

        // Reset the form after 2 seconds
        setTimeout(() => {
            successMessage.style.display = 'none';
            document.querySelector('form').reset();
        }, 2000);
    }
});
