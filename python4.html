Variables & Data Types in Python

Unlock the power of Python by understanding variables and data types, the foundation for storing and manipulating data in your programs.

Variable Declaration

In Python, variables are created by assigning a value using the = operator, with no need for explicit type declarations due to dynamic typing. 


name = "Alice"
age = 25
print(f"Name: {name}, Age: {age}")

Explanation

  • name = "Alice": Assigns a string to the variable name.
  • age = 25: Assigns an integer to age.
  • No type declaration: Python infers the type dynamically.
  • f"...": F-string formats output for display.

Numeric Types

Python supports numeric data types like integers (int) and floating-point numbers (float) for mathematical operations. 


count = 100  # Integer
price = 19.99  # Float
total = count * price
print(f"Total cost: ${total}")

Explanation

  • count = 100: Integer for whole numbers.
  • price = 19.99: Float for decimal numbers.
  • total = count * price: Performs multiplication, resulting in a float.
  • print: Displays the computed result.

String Type

Strings (str) in Python are sequences of characters, used for text manipulation, and support various operations like concatenation and slicing. 


greeting = "Hello"
name = "World"
message = greeting + ", " + name + "!"
print(message)
print(message[0:5])  # Slicing

Explanation

  • greeting = "Hello": String variable.
  • +: Concatenates strings.
  • message[0:5]: Extracts substring "Hello" using slicing.
  • Immutable: Strings cannot be modified in-place.

Boolean Type

Booleans (bool) represent True or False values, commonly used in conditional statements and logic operations. 


is_adult = True
age = 15
if age >= 18:
    is_adult = True
else:
    is_adult = False
print(f"Is adult? {is_adult}")

Explanation

  • is_adult = True: Boolean variable initialization.
  • if age >= 18: Updates boolean based on condition.
  • print: Displays the boolean value.

List Type

Lists (list) are mutable, ordered collections that can hold mixed data types, supporting operations like appending and indexing. 


fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(f"Fruits: {fruits}")
print(f"First fruit: {fruits[0]}")

Explanation

  • fruits = ["apple", "banana", "cherry"]: Creates a list.
  • fruits.append("orange"): Adds an item to the list.
  • fruits[0]: Accesses the first element via indexing.
  • Mutable: Lists can be modified after creation.

Tuple Type

Tuples (tuple) are immutable, ordered collections, ideal for fixed data sets, with similar indexing to lists. 


coordinates = (10, 20)
print(f"Coordinates: {coordinates}")
print(f"X: {coordinates[0]}")

Explanation

  • coordinates = (10, 20): Creates a tuple.
  • coordinates[0]: Accesses the first element.
  • Immutable: Tuples cannot be modified after creation.

Dictionary Type

Dictionaries (dict) store key-value pairs, allowing efficient data retrieval using unique keys. 


student = {"name": "Bob", "age": 20, "grade": "A"}
student["major"] = "Computer Science"
print(f"Student: {student}")
print(f"Name: {student['name']}")

Explanation

  • student = {"name": "Bob", ...}: Creates a dictionary.
  • student["major"] = "Computer Science": Adds a new key-value pair.
  • student['name']: Retrieves value using the key.
  • Mutable: Dictionaries can be modified.

Set Type

Sets (set) are unordered collections of unique elements, useful for operations like union and intersection. 


unique_numbers = {1, 2, 2, 3}  # Duplicates removed
unique_numbers.add(4)
print(f"Unique numbers: {unique_numbers}")

Explanation

  • unique_numbers = {1, 2, 2, 3}: Creates a set, removing duplicates.
  • unique_numbers.add(4): Adds an element to the set.
  • Unordered: Set elements have no specific order.
  • Mutable: Sets can be modified, but elements must be immutable.

Type Conversion

Python allows converting between data types using functions like int(), float(), str(), and others for flexible data handling. 


num_str = "123"
num_int = int(num_str)
num_float = float(num_str)
print(f"String: {num_str}, Integer: {num_int}, Float: {num_float}")

Explanation

  • int(num_str): Converts string to integer.
  • float(num_str): Converts string to float.
  • Explicit conversion: Ensures compatibility in operations.

Conclusion

Variables and data types are the backbone of Python programming, enabling you to store and manipulate data efficiently. With dynamic typing and a rich set of data types—integers, floats, strings, booleans, lists, tuples, dictionaries, and sets—Python offers flexibility and power. Practice using these types and type conversion to build robust programs, from simple scripts to complex applications!
