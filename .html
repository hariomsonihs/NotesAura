JavaScript Introduction

JavaScript is a versatile programming language used for web development, adding interactivity to websites.

  • Dynamic: Executes in the browser, enabling real-time updates.
  • Client-Side: Runs on the user's device, not the server.
  • Versatile: Used for front-end, back-end, and more with frameworks.

Example:

// This is a JavaScript comment
console.log("Hello, World!");
document.body.style.backgroundColor = "#f0f0f0";

Tip: Use console.log for debugging in the browser console.

Data Types and Variables

JavaScript supports various data types and variable declarations.

Type/Concept Description Example
varVariable declaration (function-scoped)var x = 10;
letBlock-scoped variablelet y = 20;
constBlock-scoped, cannot be reassignedconst z = 30;
NumberNumeric valueslet num = 42;
StringText valueslet str = "Hello";
BooleanTrue or falselet bool = true;
NullRepresents no valuelet n = null;
UndefinedVariable not assignedlet u;

Example:

let name = "NotesAura";
const PI = 3.14;
var count = 100;

console.log(typeof name); // "string"
console.log(typeof PI); // "number"
console.log(typeof count); // "number"

Fact: Use const for values that won't change to prevent accidental reassignment.

Functions

Functions are reusable blocks of code that perform specific tasks.

Type Description Example
Function DeclarationNamed functionfunction add(a, b) { return a + b; }
Function ExpressionFunction as a variableconst add = function(a, b) { return a + b; };
Arrow FunctionConcise syntax, no own thisconst add = (a, b) => a + b;
Default ParametersSets default valuesfunction greet(name = "Guest") { ... }
Rest ParametersCollects arguments into arrayfunction sum(...numbers) { ... }

Example:

function greet(name = "User") {
    return `Hello, ${name}!`;
}

const multiply = (a, b) => a * b;

function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

console.log(greet()); // "Hello, User!"
console.log(multiply(5, 3)); // 15
console.log(sum(1, 2, 3, 4)); // 10

Tip: Use arrow functions for concise one-liners and to preserve this context.

Arrays and Objects

Arrays and objects are core data structures in JavaScript.

Method/Property Description Example
Array.push()Adds element to endarr.push(4);
Array.pop()Removes last elementarr.pop();
Array.map()Creates new array with resultsarr.map(x => x * 2);
Array.filter()Filters elementsarr.filter(x => x > 2);
Object.keys()Gets object keysObject.keys(obj);
Object.values()Gets object valuesObject.values(obj);
DestructuringUnpacks valuesconst { name } = obj;

Example:

const numbers = [1, 2, 3];
numbers.push(4); // [1, 2, 3, 4]
const doubled = numbers.map(num => num * 2); // [2, 4, 6, 8]

const user = { name: "Alex", age: 25 };
const { name, age } = user;

console.log(Object.keys(user)); // ["name", "age"]
console.log(doubled); // [2, 4, 6, 8]
console.log(name); // "Alex"

Tip: Use destructuring to simplify accessing object properties and array elements.

Event Handling

Handle user interactions like clicks, keypresses, and more.

Method/Event Description Example
addEventListenerAttaches event listenerelem.addEventListener("click", fn);
clickMouse click eventbtn.addEventListener("click", () => {});
inputInput field changesinput.addEventListener("input", fn);
keydownKey press eventdocument.addEventListener("keydown", fn);
event.preventDefaultPrevents default behaviorevent.preventDefault();

Example:

const button = document.querySelector(".btn");
button.addEventListener("click", () => {
    alert("Button clicked!");
});

const input = document.querySelector("input");
input.addEventListener("input", (e) => {
    console.log(e.target.value);
});

document.addEventListener("keydown", (e) => {
    if (e.key === "Enter") e.preventDefault();
});

Fact: Use event.preventDefault() to stop default actions like form submissions.

Promises and Async/Await

Handle asynchronous operations like API calls.

Concept Description Example
PromiseRepresents future valuenew Promise((resolve, reject) => {});
.then()Handles resolved valuepromise.then(data => console.log(data));
.catch()Handles errorspromise.catch(err => console.error(err));
asyncDeclares async functionasync function fetchData() {}
awaitPauses for promise resolutionconst data = await promise;

Example:

const myPromise = new Promise((resolve, reject) => {
    setTimeout(() => resolve("Success!"), 1000);
});

myPromise
    .then(result => console.log(result))
    .catch(error => console.error(error));

async function fetchData() {
    try {
        const response = await fetch("https://api.example.com/data");
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Error:", error);
    }
}

Tip: Use try/catch with async/await for cleaner error handling.

Modules

ES6 modules organize code into reusable files.

Concept Description Example
exportExports a variable or functionexport const name = "value";
importImports from another moduleimport { name } from "./module.js";
export defaultExports default valueexport default function() {}
import asAliases imported itemsimport { name as alias } from "./module.js";

Example:

// math.js
export const add = (a, b) => a + b;
export default function multiply(a, b) {
    return a * b;
}

// main.js
import multiply, { add as sum } from "./math.js";

console.log(sum(2, 3)); // 5
console.log(multiply(4, 5)); // 20

Tip: Use modules to keep code modular and maintainable.
