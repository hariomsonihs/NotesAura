Basic Syntax & Structure of C++

Unlock the fundamentals of C++ programming by mastering its syntax and structure. This guide introduces the building blocks of C++ with clear examples and explanations.

Program Structure

A C++ program follows a specific structure, including headers, namespaces, and a main function, as shown in this classic "Hello, World!" example. 


#include 
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Explanation

  • #include : Includes the input/output stream library.
  • using namespace std;: Allows use of standard library names like cout.
  • int main(): The programâ€™s entry point.
  • cout <<: Outputs text to the console.
  • return 0;: Indicates successful execution.

Variables and Data Types

Variables store data, and C++ supports various data types like integers, floats, and strings to handle different kinds of information. 


#include 
#include 
using namespace std;

int main() {
    int age = 25;
    double salary = 55000.50;
    string name = "Alice";
    cout << "Name: " << name << ", Age: " << age << ", Salary: " << salary << endl;
    return 0;
}

Explanation

  • int: Stores whole numbers (e.g., age).
  • double: Stores decimal numbers (e.g., salary).
  • string: Stores text (e.g., name).
  • =: Assigns values to variables.

Control Structures

Control structures like conditionals (if) and loops (for, while) direct the flow of a program. 


#include 
using namespace std;

int main() {
    int num = 10;
    if (num > 0) {
        cout << num << " is positive" << endl;
    }
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    cout << endl;
    return 0;
}

Explanation

  • if: Executes code if a condition is true.
  • for: Repeats code for a specified number of iterations.
  • cout: Displays output within control structures.

Functions

Functions are reusable blocks of code that perform specific tasks, improving modularity and readability. 


#include 
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    int x = 5, y = 3;
    int sum = add(x, y);
    cout << "Sum of " << x << " and " << y << ": " << sum << endl;
    return 0;
}

Explanation

  • int add(int a, int b): Defines a function with two parameters.
  • return: Sends back the result to the caller.
  • sum = add(x, y): Calls the function with arguments.

Input/Output

C++ uses cin for input and cout for output, enabling interaction with users via the console. 


#include 
using namespace std;

int main() {
    string name;
    cout << "Enter your name: ";
    getline(cin, name);
    cout << "Hello, " << name << "!" << endl;
    return 0;
}

Explanation

  • cout <<: Outputs text to the console.
  • getline(cin, name): Reads a full line of input, including spaces.
  • endl: Adds a newline and flushes the output buffer.
