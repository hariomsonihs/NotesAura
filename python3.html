Variables and Data Types

Python supports dynamic typing, allowing variables to hold various data types like integers, floats, strings, and lists without explicit declarations. 


name = "Alice"  # String
age = 25  # Integer
salary = 55000.50  # Float
skills = ["Python", "Java"]  # List
print(f"Name: {name}, Age: {age}, Salary: {salary}, Skills: {skills}")

Explanation

  • name = "Alice": Assigns a string without type declaration.
  • age = 25: Integer variable.
  • salary = 55000.50: Float for decimal numbers.
  • skills = ["Python", "Java"]: List to store multiple values.
  • f"...": F-string for formatted output.

Control Structures

Python uses conditionals (if) and loops (for, while) to control program flow, with indentation defining scope. 


num = 10
if num > 0:
    print(f"{num} is positive")
else:
    print(f"{num} is non-positive")

for i in range(1, 6):
    print(i, end=" ")
print()  # Newline

Explanation

  • if num > 0:: Conditional statement with colon and indented block.
  • for i in range(1, 6):: Loops from 1 to 5.
  • end=" ": Customizes print output to avoid newlines.
  • Indentation: Defines the scope of if and for blocks.

Functions

Functions in Python are defined using the def keyword, enabling reusable code blocks for specific tasks. 


def add(a, b):
    return a + b

x, y = 5, 3
result = add(x, y)
print(f"Sum of {x} and {y} is: {result}")

Explanation

  • def add(a, b):: Defines a function with two parameters.
  • return a + b: Returns the computed sum.
  • result = add(x, y): Calls the function with arguments.
  • F-string: Formats output for readability.

Input/Output

Python uses input() for user input and print() for output, enabling interactive console applications. 


name = input("Enter your name: ")
print(f"Hello, {name}!")

Explanation

  • input(): Captures user input as a string.
  • print(f"Hello, {name}!"): Outputs formatted text.
  • Simple I/O: Facilitates user interaction in scripts.
