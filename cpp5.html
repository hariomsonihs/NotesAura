Classes & Objects in C++

Dive into the core of object-oriented programming in C++ by mastering classes and objects, the building blocks for creating modular and reusable code.

What are Classes?

A class is a user-defined blueprint that defines properties (data members) and behaviors (member functions) for objects, enabling structured programming. 


#include 
using namespace std;

class Rectangle {
public:
    double length;
    double width;
    double area() {
        return length * width;
    }
};

int main() {
    Rectangle rect;
    rect.length = 5.0;
    rect.width = 3.0;
    cout << "Area: " << rect.area() << endl;
    return 0;
}

Explanation

  • class Rectangle: Defines a class with data members (length, width) and a method (area).
  • public:: Allows access to class members from outside the class.
  • rect.area(): Calls the method to compute the area.

Creating Objects

Objects are instances of a class, created to use the classâ€™s properties and methods in a program. 


#include 
using namespace std;

class Student {
public:
    string name;
    int rollNo;
    void display() {
        cout << "Name: " << name << ", Roll No: " << rollNo << endl;
    }
};

int main() {
    Student s1, s2;
    s1.name = "Alice";
    s1.rollNo = 101;
    s2.name = "Bob";
    s2.rollNo = 102;
    s1.display();
    s2.display();
    return 0;
}

Explanation

  • Student s1, s2: Creates two objects of the Student class.
  • s1.name: Assigns values to object properties.
  • s1.display(): Calls the method for each object.

Access Specifiers

Access specifiers (public, private, protected) control the visibility of class members, ensuring data encapsulation. 


#include 
using namespace std;

class BankAccount {
private:
    double balance;
public:
    void setBalance(double amount) {
        if (amount >= 0) balance = amount;
    }
    double getBalance() {
        return balance;
    }
};

int main() {
    BankAccount account;
    account.setBalance(500.75);
    cout << "Balance: $" << account.getBalance() << endl;
    return 0;
}

Explanation

  • private: double balance: Restricts direct access to the data member.
  • public: setBalance: Provides controlled access to modify data.
  • getBalance(): Allows safe retrieval of the private member.

Constructors

Constructors are special member functions that initialize objects when they are created, with optional parameters for customization. 


#include 
using namespace std;

class Book {
public:
    string title;
    int pages;
    Book(string t, int p) {
        title = t;
        pages = p;
    }
    void display() {
        cout << "Book: " << title << ", Pages: " << pages << endl;
    }
};

int main() {
    Book book1("C++ Primer", 1200);
    book1.display();
    return 0;
}

Explanation

  • Book(string t, int p): Defines a parameterized constructor.
  • Book book1("C++ Primer", 1200): Creates an object with initialized values.
  • display(): Shows the initialized data.

Destructors

Destructors are special functions that clean up resources when an object goes out of scope, ensuring proper memory management. 


#include 
using namespace std;

class Resource {
public:
    Resource() {
        cout << "Resource allocated" << endl;
    }
    ~Resource() {
        cout << "Resource deallocated" << endl;
    }
};

int main() {
    {
        Resource res;
    } // Destructor called when res goes out of scope
    cout << "After scope" << endl;
    return 0;
}

Explanation

  • ~Resource(): Defines the destructor, called automatically.
  • { Resource res; }: Creates an object within a scope.
  • Destructor output: Triggered when the objectâ€™s scope ends.

Practical Applications

Classes and objects are used to model real-world entities in applications like games, simulations, and software systems.

  • Game Development: Classes for characters, items, and environments.
  • Database Systems: Objects to represent records and queries.
  • GUI Applications: Classes for buttons, windows, and controls.
  • Simulations: Modeling physical systems with object interactions.

Classes and objects are the heart of object-oriented programming in C++, enabling you to create structured, reusable, and maintainable code.
