CSS Introduction

CSS (Cascading Style Sheets) is used to style and layout web pages.

  • Cascading: Styles cascade from top to bottom.
  • Specificity: Determines which styles are applied when conflicts occur.
  • Inheritance: Some properties inherit values from parent elements.

Example:

/* This is a CSS comment */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

Tip: Always use external CSS files for better maintainability.

CSS Selectors

Selectors target HTML elements to apply styles.

Selector Example Description
ElementpSelects all <p> elements
Class.introSelects all elements with class="intro"
ID#headerSelects the element with id="header"
Universal*Selects all elements
Attribute[target]Selects elements with a target attribute
Pseudo-classa:hoverSelects links on mouse hover

Example:

/* Element selector */
p {
    color: blue;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* ID selector */
#main-content {
    padding: 20px;
}

/* Pseudo-class */
button:hover {
    background-color: #4f46e5;
}

Fact: ID selectors have higher specificity than class selectors.

CSS Box Model

Every element is a rectangular box with these properties:

  • Content: The actual content of the box
  • Padding: Clears an area around the content
  • Border: A border that goes around the padding
  • Margin: Clears an area outside the border
Property Description Example
widthSets the width of an elementwidth: 300px;
heightSets the height of an elementheight: 200px;
paddingSets space inside the elementpadding: 10px;
marginSets space outside the elementmargin: 20px;
borderSets the border around the elementborder: 1px solid black;
box-sizingControls how width/height is calculatedbox-sizing: border-box;

Example:

.box {
    width: 300px;
    padding: 20px;
    border: 5px solid gray;
    margin: 10px;
    box-sizing: border-box; /* includes padding and border in width */
}

Trick: Use box-sizing: border-box for more predictable sizing.

Flexbox Layout

Flexbox provides efficient way to layout, align and distribute space among items.

Property Description Example
display: flexCreates a flex containerdisplay: flex;
flex-directionSets direction of flex itemsflex-direction: row;
flex-wrapControls wrapping of itemsflex-wrap: wrap;
justify-contentAligns items along main axisjustify-content: center;
align-itemsAligns items along cross axisalign-items: center;
align-contentAligns flex linesalign-content: space-between;
flex-growDefines ability to growflex-grow: 1;
flex-shrinkDefines ability to shrinkflex-shrink: 0;
flex-basisDefault size before remaining spaceflex-basis: 100px;
orderChanges order of itemsorder: 2;

Example:

.container {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
    justify-content: space-between;
    align-items: center;
    gap: 20px;
}

.item {
    flex: 1 1 200px; /* grow shrink basis */
    order: 1;
}

Tip: Use gap property for consistent spacing between flex items.

Grid Layout

CSS Grid provides a two-dimensional layout system.

Property Description Example
display: gridCreates a grid containerdisplay: grid;
grid-template-columnsDefines columnsgrid-template-columns: 1fr 1fr 1fr;
grid-template-rowsDefines rowsgrid-template-rows: 100px auto;
grid-template-areasDefines template areasgrid-template-areas: "header header" "sidebar main";
grid-gapSets gap between itemsgap: 20px;
grid-columnPositions item in columnsgrid-column: 1 / 3;
grid-rowPositions item in rowsgrid-row: 1;
grid-areaAssigns item to areagrid-area: header;
justify-itemsAligns items along row axisjustify-items: center;
align-itemsAligns items along column axisalign-items: start;

Example:

.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: 80px auto 100px;
    grid-template-areas: 
        "header header"
        "sidebar main"
        "footer footer";
    gap: 15px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Fact: Grid is perfect for complex layouts while Flexbox is better for one-dimensional layouts.

Transitions & Animations

Create smooth transitions and animations between states.

Property Description Example
transitionShorthand for transition propertiestransition: all 0.3s ease;
transition-propertySpecifies which properties to animatetransition-property: width;
transition-durationHow long transition takestransition-duration: 0.5s;
transition-timing-functionSpeed curve of transitiontransition-timing-function: ease-in;
transition-delayWhen transition startstransition-delay: 0.2s;
animationShorthand for animation propertiesanimation: slide 2s infinite;
@keyframesDefines the animation sequence@keyframes slide { from { transform: translateX(0); } }
animation-nameName of @keyframes animationanimation-name: slide;
animation-durationHow long animation takesanimation-duration: 3s;
animation-timing-functionSpeed curve of animationanimation-timing-function: linear;
animation-iteration-countHow many times animation playsanimation-iteration-count: infinite;

Example:

/* Transition example */
.button {
    background-color: #4f46e5;
    transition: background-color 0.3s ease, transform 0.2s;
}

.button:hover {
    background-color: #6366f1;
    transform: translateY(-2px);
}

/* Animation example */
@keyframes slideIn {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
}

.slide-in {
    animation: slideIn 0.5s ease-out forwards;
}

Tip: Use will-change property to hint browsers about upcoming animations for better performance.

Responsive Design

Techniques to make websites work on all device sizes.

Concept Description Example
Viewport Meta TagControls mobile layout<meta name="viewport" content="width=device-width">
Media QueriesConditional CSS rules@media (max-width: 768px) { ... }
Fluid LayoutsUses % rather than pxwidth: 100%; max-width: 1200px;
Flexible ImagesImages that scaleimg { max-width: 100%; }
Mobile-firstDesign for mobile first@media (min-width: 768px) { ... }
Relative Unitsem, rem, vw, vhfont-size: 1.2rem;

Example:

/* Mobile-first approach with media queries */
body {
    font-size: 16px;
    width: 100%;
}

img {
    max-width: 100%;
    height: auto;
}

.container {
    max-width: 1200px;
    margin: 0 auto;
}

@media (min-width: 768px) {
    body {
        font-size: 18px;
    }
    .container {
        width: 90%;
    }
}

Tip: Always include the viewport meta tag for proper mobile scaling.

Pseudo-Elements

Style specific parts of an element without adding extra markup.

Pseudo-Element Description Example
::beforeInserts content before an elementp::before { content: "★"; }
::afterInserts content after an elementp::after { content: "★"; }
::first-lineStyles the first line of textp::first-line { font-weight: bold; }
::first-letterStyles the first letter of textp::first-letter { font-size: 2em; }
::selectionStyles selected text::selection { background: #4f46e5; }

Example:

.quote::before {
    content: '"';
    font-size: 1.5rem;
    color: #4f46e5;
    margin-right: 5px;
}

.quote::after {
    content: '"';
    font-size: 1.5rem;
    color: #4f46e5;
    margin-left: 5px;
}

p::first-letter {
    font-size: 2rem;
    font-weight: bold;
    float: left;
    margin-right: 5px;
}

::selection {
    background-color: #8b5cf6;
    color: white;
}

Fact: Pseudo-elements require double colons (::) in modern CSS.

CSS Variables

Custom properties for reusable values across your CSS.

Concept Description Example
--variableDefines a custom property--main-color: #4f46e5;
var()Uses a custom propertycolor: var(--main-color);
:rootGlobal scope for variables:root { --spacing: 10px; }
FallbackProvides default valuecolor: var(--main-color, #000);

Example:

:root {
    --primary-color: #4f46e5;
    --secondary-color: #8b5cf6;
    --spacing: 15px;
}

.button {
    background-color: var(--primary-color);
    padding: var(--spacing);
}

.card {
    border: 2px solid var(--secondary-color);
    margin: var(--spacing, 10px);
}

Tip: Define variables in :root for global access and easy maintenance.
