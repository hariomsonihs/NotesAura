Control Flow in Python

Master Python’s control flow tools to direct the execution of your programs using conditionals, loops, and control statements, enabling dynamic and efficient code.

Conditional Statements

Conditional statements (if, elif, else) allow you to execute code based on specific conditions, using indentation to define scope. 


score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"
print(f"Score: {score}, Grade: {grade}")

Explanation

  • if score >= 90:: Checks the first condition, executes if true.
  • elif score >= 80:: Checks subsequent conditions if prior ones fail.
  • else:: Executes if no conditions are met.
  • Indentation: Defines the block of code for each branch.
  • Logical operators: Can use and, or, not for complex conditions.

For Loop

The for loop iterates over a sequence (e.g., list, range, string), executing a block of code for each element. 


fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"Fruit: {fruit}")
total = 0
for num in range(1, 6):
    total += num
print(f"Sum of 1 to 5: {total}")

Explanation

  • for fruit in fruits:: Iterates over each element in the list.
  • range(1, 6): Generates numbers from 1 to 5 (excludes 6).
  • total += num: Accumulates sum during iteration.
  • Indentation: Defines the loop’s code block.
  • Use cases: Ideal for iterating over known sequences or fixed ranges.

While Loop

The while loop executes a block of code as long as a condition remains true, suitable for indefinite iterations. 


count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1

Explanation

  • while count <= 5:: Continues loop while condition is true.
  • count += 1: Increments counter to avoid infinite loop.
  • Indentation: Defines the loop’s code block.
  • Caution: Ensure condition eventually becomes false to prevent infinite loops.
  • Use cases: Useful when the number of iterations is unknown.

Loop Control Statements

Loop control statements (break, continue, pass) modify loop behavior, allowing precise control over iteration. 


# Break example
for num in range(1, 10):
    if num == 5:
        break
    print(f"Number: {num}")

# Continue example
for num in range(1, 6):
    if num % 2 == 0:
        continue
    print(f"Odd number: {num}")

# Pass example
for num in range(1, 4):
    if num == 2:
        pass  # Placeholder, does nothing
    print(f"Number: {num}")

Explanation

  • break: Exits the loop immediately when num == 5.
  • continue: Skips the rest of the loop body for even numbers, continuing with the next iteration.
  • pass: Acts as a placeholder, allowing empty blocks without errors.
  • Use cases: break for early termination, continue for skipping iterations, pass for future implementation.

Nested Control Structures

Nested control structures involve placing conditionals or loops inside others, enabling complex logic and pattern generation. 


# Nested loops for a multiplication table
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} * {j} = {i * j}")
    print()  # Newline after each row

# Nested conditional and loop
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    if num % 2 == 0:
        print(f"{num} is even")
    else:
        print(f"{num} is odd")

Explanation

  • for i in range(1, 4):: Outer loop for rows.
  • for j in range(1, 4):: Inner loop for columns, generating multiplication table.
  • Nested if: Checks even/odd within a loop.
  • Indentation: Critical for defining nested block boundaries.
  • Use cases: Patterns, matrix operations, hierarchical decision-making.

Practical Example

This example combines conditionals, loops, and control statements to create a number guessing game, showcasing real-world control flow application. 


import random

secret = random.randint(1, 10)
attempts = 3

while attempts > 0:
    guess = int(input("Guess a number (1-10): "))
    if guess == secret:
        print("Congratulations! You guessed it!")
        break
    elif guess < secret:
        print("Too low!")
    else:
        print("Too high!")
    attempts -= 1
    if attempts == 0:
        print(f"Game over! The number was {secret}")

Explanation

  • while attempts > 0:: Loops until attempts are exhausted.
  • if guess == secret:: Checks for correct guess, breaks loop if true.
  • elif guess < secret:: Provides feedback for incorrect guesses.
  • attempts -= 1: Decrements attempts each iteration.
  • break: Exits loop on correct guess, avoiding unnecessary iterations.
  • Real-world use: Interactive applications, games, decision systems.

Conclusion

Control flow in Python, with its intuitive conditionals, versatile loops, and powerful control statements, enables you to craft dynamic and efficient programs. By mastering if, for, while, break, continue, pass, and nested structures, you can handle complex logic and build interactive applications. Practice these constructs through examples like the number guessing game to unlock Python’s full potential!
